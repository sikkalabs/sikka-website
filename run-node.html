Batteries Included

Everything in one binary

Web Wallet

Full-featured browser wallet served locally. Create keys, send, receive, and manage multisig — without trusting any third party.

DAG Explorer

Real-time transaction explorer with live DAG visualization. Browse the graph, inspect any transaction, and monitor network health.

Full DAG Engine

Complete validation, consensus, and propagation engine. Syncs with peers, validates all transactions, and participates in the network.

Managed Tor

Built-in Tor integration auto-configures a .onion address for your node. No separate Tor setup needed — privacy is on by default.

HTTP API

Clean REST API with CORS support. Build custom applications, integrate with existing services, or federate with other nodes.

Quick Setup

Up in three steps

1

Install Docker

If you don't have Docker, install it from docs.docker.com/get-docker. Available for Windows, macOS, and Linux.

2

Run the container

Copy and run the command below. The image is ~17 MB and will start syncing immediately.

docker run -d --restart unless-stopped \
  -p 64552:64552 \
  -v sikka-data:/home/sikka/data \
  ghcr.io/sikkalabs/sikka:latest
3

Open your browser

Navigate to http://localhost:64552. You'll see the wallet and DAG explorer running locally. You're a node operator!
